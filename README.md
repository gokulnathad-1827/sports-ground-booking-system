# Sports Ground Booking System

## About

The Sports Ground Booking System is a Java console application that allows users to book sports grounds like football turfs, cricket grounds, and badminton courts.

This project is developed using Java, Object-Oriented Programming (OOP), ArrayList, and Exception Handling.

## Features

- Register users
- View available grounds
- Book a sports ground
- Check slot availability
- Calculate booking fare
- Cancel booking
- View user bookings
- View ground schedule
- Custom exception handling

## Technologies Used

- Java
- Eclipse IDE
- ArrayList
- OOP Concepts
- Exception Handling
